retdec安装使用

安装(Linux)

环境要求

  • sudo apt-get install build-essential cmake git openssl libssl-dev python3 autoconf automake libtool pkg-config m4 zlib1g-dev upx doxygen graphviz

编译安装

  • git clone https://github.com/avast/retdec
  • cd retdec
  • mkdir build && cd build
  • cmake .. -DCMAKE_INSTALL_PREFIX=/usr/local/retdec(单独指定一个目录，不要使用默认的安装目录)
  • make -jN (N 多核编译)
  • make install

使用入门

拿hello world试一下

  • gcc test.c
  • /usr/local/retdec/bin/retdec-decompiler a.out

可以看到产生了如下文件

  • input.exe.dsm

    ：以作者自定义格式反汇编输出。指令助记符采用默认的顶点格式。

  • input.exe.bc

    ：核心反编译部分的最终产品，采用 LLVM bitcode格式。

  • input.exe.ll

    ：以 LLVM IR 格式对 LLVM bitcocde进行人类可读的反汇编。

  • input.exe.config.json

    ：反编译过程生成的元数据。

  • input.exe.c

    ：反编译的 C 代码。这是主要输出。

RetDec 可以分为三个阶段：

  • 预处理阶段：首先检查文件类型是否为可执行文件，然后调用 fileinfo 获取文件信息生成 a.out.c.json，然后调用 unpacker 查壳和脱壳等操作
  • 核心阶段：接下来才是重头戏，调用 bin2llvmir 将二进制文件转换成 LLVM IR，并输出 a.out.c.frontend.dsm、a.out.c.backend.ll 和 a.out.c.backend.bc
  • 后端阶段：这个阶段通过一系列代码优化和生成等操作，将 LLVM IR 反编译成 C 代码 a.out.c，还有 CFG 等。
